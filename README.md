# API Testing Project - GoRest (Postman)

## Project Overview

This project demonstrates API testing using Postman on the GoRest API. It includes CRUD operations, request chaining, environment variables, and negative testing scenarios.


## Tools Used

* Postman
* GoRest API
* Collection Runner
* Environment Variables


## Test Scenarios

### CRUD Operations

* Create User (POST)
* Get User (GET)
* Update User (PUT)
* Delete User (DELETE)

### Negative Testing

* Duplicate Email Test
* Invalid User ID Test
* Missing Token Test
* Missing Required Fields Test
* Invalid Email Format Test


## Concepts Covered

* REST API Testing
* HTTP Methods
* Request Chaining (user_id)
* Environment Variables
* Assertions in Postman
* Collection Runner Execution


## How to Run

1. Import collection file into Postman
2. Import environment file
3. Select environment
4. Run collection using Collection Runner


## Expected Results

* CRUD Tests → PASS (200/201/204)
* Negative Tests → PASS (expected errors like 401/404/422)
